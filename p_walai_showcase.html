Walai Showcase

Product:  Walaito |Platform:  Desktop/Tablet/Mobile

This is my first personal website where I showcase the outstanding projects within digital design, UX/UI design, web development and multimedia design. I created the website from scratch using the design thinking process, and web development knowledge.

The mood and tone of the website is simple as I wanted it look clean and ease of use. The picture and graphics mostly are made by using Photoshop and Illustrator while the layout and sitemap are prototyped by Figma, and the development part is implemented by HTML, CSS and JS. I use Git as a version control system so that I can keep track of the changes. The website is hosted on GitHub as well.

The website also focus on SEO-friendly, which it gets the high score from Lighthouse, an open-source, automated tool for improving the quality of web pages. It has audits for performance, accessibility, progressive web apps, SEO and more.

The video where it shows the screen devices, I composited the screen-captured footage and the device mockup together by using Adobe After Effects.

This project has shown how I use my multimedia skills to create all the graphic materials, and I use the knowledge in web development to implement the website. I combined my competencies and knowledge to solve the issues that happened along the process, this made me realise of self-developed because technolygy is growing rapidly so I better prepare for the future and coming new technologies as well as design trend.

Last but not least, since I have mentioned that this project is my first personal website, so of course the website you are looking at right now is a new one, there is more improvment from the first one, and there will more new technologies and trend that I will keep my eyes on it to make my work even better.
